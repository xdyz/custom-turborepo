### 简介 

注意是 类别（kind）而不是 类型（type）

类别是一个大范围的概念，类型是某个类别下的 一种

例如 type Student struct  struct 是类别，Student 是自定义类型 属于struct 类别下。

类别 如下：

- Invalid Kind = iota
- Bool
- Int
- Int8
- Int16
- Int32
- Int64
- Uint
- Uint8
- Uint16
- Uint32
- Uint64
- Uintptr
- Float32
- Float64
- Complex64
- Complex128
- Array
- Chan
- Func
- Interface
- Map
- Pointer
- Slice
- String
- Struct
- UnsafePointer
